To get the functionality done you need to setup your personal site. Right now we don't provide more than 3 input fields e.g. Name, Email, Message. You need to give proper id to each input field. You'll need jQuery to connect with the server. Copy the code provided by us from the collapsible element and paste it in your script tag. In name, email, and message field please set the id name you used in your form.
