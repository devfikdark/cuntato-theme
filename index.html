CUNTATO

No more showcasing non-functional contact form in static websites. Give life to your static website's contact form by adding the back-end code which really works!

Get started

Easy Authentication

You don't need to create any account in our site. To get started you need to authenticate via Google or Github or LinkedIn.

Setup

After authentication you'll be redirected to your dashboard page where you will create project for a single contact form. You'll be given required credentials after creating the project.

Get messages

Use the required credentials provided after creating project in your site's script and BOOM!!! You are good to go.
