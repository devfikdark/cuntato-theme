EASY CONTACT

Do you have any personal site or any project which has contact form? But your contact form doesn't work!!! If you are looking for a code which gives your contact form life then you are in the right place.

Learn more

Authentication

You don't need to create any account in our site. To get started you need to authenticate via Google or Github or Twitter.

Setup

After authentication you'll be redirected to your profile page where you will create a project for a single contact form. You'll be given required credentials after creating the project.

Get started

Use the required credentials provided after creating project in your site's script and BOOM!!! You are good to go.
